# Partner logos

Drop each partner's logo here using these EXACT filenames (PNG with transparent background works best, ~200px wide):

- `tn-govt.png`  -> Govt of Tamil Nadu
- `ssf-plastic.png`  -> SSF Plastic India
- `global-calcium.png`  -> Global Calcium
- `ola.png`  -> OLA
- `gail.png`  -> GAIL (India) Ltd
- `tata-steel.png`  -> TATA Steel
- `bimetal-bearings.png`  -> Bimetal Bearings
- `re-sustainability.png`  -> RE Sustainability
- `credai.png`  -> CREDAI
- `modi-granite.png`  -> Modi Granite

If a file is missing, the site automatically falls back to showing the company name as text. SVG also works — if you use SVG, change the extension in index.html accordingly.
